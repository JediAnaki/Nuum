# 101 - Плейлисты

**Приоритет:** Высокий (101)
**Категория:** Организация контента
**Зависимости:** 003 (Главная), 004 (Страница видео)

## Описание

Система создания, редактирования и просмотра плейлистов для организации и курирования видео контента.

## Функциональные требования

### 1. Просмотр плейлиста

**URL:** `/playlist?list={playlist_id}`

**Структура:**
```
┌─────────────────────────────────────────────┐
│ ПЛЕЕР ВИДЕО (16:9)                          │
├─────────────────────────────────────────────┤
│ Название видео                              │
│ Канал • просмотры • дата                    │
└─────────────────────────────────────────────┘

┌──────────────────────────────────────┐
│ Название плейлиста                   │
│ Автор плейлиста                      │
│ 12 видео • 50 просмотров             │
│ Последнее обновление: 2 дня назад    │
│                                      │
│ Описание плейлиста...                │
│                                      │
│ [▶ Воспроизвести все]               │
│ [🔀 Перемешать] [➕ Сохранить]       │
│                                      │
│ ☑ 1. [Thumbnail] Название 1    5:30 │ ← текущее
│ ☐ 2. [Thumbnail] Название 2   10:20 │
│ ☐ 3. [Thumbnail] Название 3    3:45 │
│ ...                                  │
└──────────────────────────────────────┘
```

**Функции:**
- Воспроизвести все с первого
- Перемешать порядок
- Сохранить плейлист себе
- Автопереход к следующему видео
- Отметки просмотренных

### 2. Создание плейлиста

**Модальное окно:**
```
┌───────────────────────────────┐
│ Новый плейлист           [X]  │
├───────────────────────────────┤
│ Название:                     │
│ [_______________________]     │
│                               │
│ Описание:                     │
│ [_______________________]     │
│ [_______________________]     │
│                               │
│ Видимость:                    │
│ ○ Публичный                   │
│ ● По ссылке                   │
│ ○ Приватный                   │
│                               │
│ [Отмена]         [Создать]   │
└───────────────────────────────┘
```

### 3. Редактирование плейлиста

**Страница редактирования:** `/playlist/{id}/edit`

**Функции:**
- Изменить название и описание
- Изменить видимость
- Изменить порядок (drag & drop)
- Добавить видео (поиск + кнопка)
- Удалить видео из плейлиста
- Дублировать плейлист
- Удалить плейлист

### 4. Добавление в плейлист

**Из страницы видео - кнопка "Сохранить":**
```
┌────────────────────────────┐
│ Сохранить в...        [X]  │
├────────────────────────────┤
│ ☑ Смотреть позже           │
│ ☐ Избранное                │
│ ☑ Мой плейлист игр         │
│ ☐ Обучение                 │
│                            │
│ [+ Создать плейлист]      │
└────────────────────────────┘
```

### 5. Типы плейлистов

#### 5.1 Пользовательские
- Созданные пользователем
- Редактируемые
- Публичные/приватные

#### 5.2 Системные (автоматические)
- **Смотреть позже** - ручное добавление
- **Понравилось** - автоматически лайкнутые
- **История** - автоматически просмотренные

#### 5.3 Плейлисты каналов
- Созданные автором канала
- Публичные для всех
- Категоризация контента

### 6. Мои плейлисты

**URL:** `/playlists` или `/library/playlists`

**Сетка плейлистов:**
```
[ Мои плейлисты ] [ Сохраненные ]

[Плейлист 1      ] [Плейлист 2      ]
[12 видео • 2дн  ] [5 видео • нед   ]
[Публичный       ] [Приватный       ]
[✏️] [🗑️]         [✏️] [🗑️]
```

### 7. Настройки плейлиста

- **Автодобавление** - автоматически добавлять видео по критериям
- **Сортировка** - по дате, популярности, вручную
- **Коллаборация** - разрешить другим добавлять (для публичных)

## Технические детали

### API endpoints
```typescript
GET /api/v1/playlists/{id}
POST /api/v1/playlists
PATCH /api/v1/playlists/{id}
DELETE /api/v1/playlists/{id}

POST /api/v1/playlists/{id}/videos
DELETE /api/v1/playlists/{id}/videos/{video_id}
PATCH /api/v1/playlists/{id}/reorder

GET /api/v1/users/me/playlists
```

### Типы данных
```typescript
interface Playlist {
  id: string;
  title: string;
  description?: string;
  visibility: "public" | "unlisted" | "private";
  thumbnail?: string; // первое видео
  video_count: number;
  view_count: number;
  created_at: string;
  updated_at: string;
  owner: {
    id: string;
    username: string;
    avatar: string;
  };
  videos: PlaylistVideo[];
}

interface PlaylistVideo {
  id: string;
  position: number;
  video: Video;
  added_at: string;
}
```

## Метрики успеха
- Плейлистов на активного пользователя > 5
- Видео в плейлисте в среднем > 8
- Watch-through rate плейлистов > 40%
- Конверсия создания плейлиста > 20%
