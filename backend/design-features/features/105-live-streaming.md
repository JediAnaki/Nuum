# 105 - Прямые трансляции (Live Streaming)

**Приоритет:** Высокий (105)
**Категория:** Контент
**Зависимости:** 002 (Плеер), 008 (Канал)

## Описание
Функциональность прямых эфиров с чатом, донатами и интерактивными элементами.

## Запуск стрима

### Методы
1. **Через Studio** - `/studio/live`
2. **Streaming software** - OBS, Streamlabs (RTMP)
3. **Мобильное приложение** (будущее)

### Настройка стрима
```
Название стрима: [____________]
Описание: [____________]
Категория: [Игры ▼]
Видимость: ○ Публичный ● По ссылке ○ Приватный
Планирование: [📅 Начать сейчас ▼]

Stream key: ***********[👁][📋]
Server URL: rtmp://platform.ru/live

[Начать трансляцию]
```

## Просмотр стрима

### Плеер
- Badge "🔴 LIVE" (пульсирующий)
- Задержка 3-10 секунд (Low Latency)
- Качество адаптируется к скорости
- Количество зрителей в реальном времени

### Чат
```
┌────────────────────────┐
│ 🔴 LIVE • 1.2K зрителей│
├────────────────────────┤
│ @user1: Привет!        │
│ @user2: 👋             │
│ @user3 💰: Донат 100р  │
│ ...                    │
├────────────────────────┤
│ [Сообщение...    ] [>] │
└────────────────────────┘
```

**Функции чата:**
- Real-time сообщения
- Эмодзи и стикеры
- Slow mode (ограничение частоты)
- Модерация (тайм-ауты, баны)
- Закрепление сообщений
- Только для подписчиков (опция)

## Интерактивные элементы

### Донаты (Super Chat/Stickers)
- Платные сообщения ($1-$500)
- Подсвечиваются в чате
- Закрепляются на время

### Polls (опросы)
- Создание опросов во время эфира
- Голосование зрителей
- Результаты в реальном времени

## Технические требования

### Стриминг
- **Протокол:** RTMP для входа
- **Выход:** HLS/DASH для просмотра
- **Кодеки:** H.264, AAC
- **Bitrate:** 2500-9000 Kbps
- **Resolution:** до 1080p60

### Low Latency
- WebRTC (LL-HLS) для минимальной задержки
- < 5 секунд задержки

## API
```typescript
POST /api/v1/live/start
POST /api/v1/live/end
GET /api/v1/live/{id}
GET /api/v1/live/{id}/chat
POST /api/v1/live/{id}/chat/message

// WebSocket для чата
ws://platform.ru/ws/live/{id}/chat
```

## Метрики
- Средняя длительность стрима > 1 час
- Пиковые зрители / средние > 2
- Engagement в чате > 10% зрителей
- Conversion в донаты > 1%
