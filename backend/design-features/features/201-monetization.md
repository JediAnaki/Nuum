# 201 - Монетизация

**Приоритет:** Средний (201)
**Категория:** Бизнес
**Зависимости:** 008 (Канал), 100 (Studio)

## Описание
Система монетизации для авторов через рекламу, донаты и членство.

## Типы монетизации

### 1. Реклама
- Pre-roll (перед видео)
- Mid-roll (в середине, от 8 мин)
- Post-roll (в конце)
- Display ads (баннеры)
- Overlay ads

### 2. Членство канала
- Месячная подписка ($0.99-$49.99)
- Эксклюзивные значки
- Эмодзи для членов
- Эксклюзивный контент

### 3. Super Chat/Stickers
- Платные сообщения в стримах
- Выделение сообщения
- Закрепление на время

### 4. Merchandise Shelf
- Продажа мерча прямо под видео
- Интеграция с партнерами

## Требования
- 1000+ подписчиков
- 4000+ часов просмотра за 12 месяцев
- Соответствие правилам

## Панель монетизации
`/studio/monetization`
- Статистика заработка
- График доходов
- Методы вывода
- Налоговая информация

## API
```typescript
GET /api/v1/studio/revenue
GET /api/v1/studio/revenue/transactions
POST /api/v1/studio/revenue/payout
```
