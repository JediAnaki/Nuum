# 004 - Страница просмотра видео

**Приоритет:** MVP (004)
**Категория:** Контент и взаимодействие
**Зависимости:** 001 (Авторизация), 002 (Видеоплеер), 003 (Главная страница)

## Описание

Полноценная страница просмотра видео со всеми элементами взаимодействия: плеер, информация о видео, лайки, комментарии, рекомендации. Центральная страница платформы.

## Функциональные требования

### 1. Структура страницы

#### 1.1 Маршрут
```
/watch?v={video_id}
/watch/{video_id} (альтернативный, SEO-friendly)
```

### 2. Видеоплеер

Используется компонент из фичи #002 (Видеоплеер).

**Особенности на странице видео:**
- Плеер занимает основную область
- Автовоспроизведение при загрузке страницы (опция)
- Сохранение прогресса просмотра каждые 5 секунд
- Возобновление с места остановки
- Переход к следующему видео при завершении
- Качество восстанавливается из настроек

### 3. Информация о видео

#### 3.1 Заголовок и метаданные

**Название видео:**
- Крупный шрифт (18-20px)
- Bold
- Полное название без обрезки
- Несколько строк при необходимости

**Статистика:**
```
1,234,567 просмотров • 15 июня 2024 г.
```
- Форматирование чисел с запятыми/пробелами
- Полная дата или относительная ("2 дня назад" до 7 дней)
- Серый цвет текста

**Хештеги (если есть):**
```
#игры #стрим #майнкрафт
```
- Синие кликабельные ссылки
- Ведут на страницу поиска по хештегу
- До 5 хештегов отображается

#### 3.2 Панель действий

**Кнопки (слева направо):**

1. **Лайк** 👍
   - Иконка лайка + счетчик
   - Форматирование: 123K, 1.2M
   - Активное состояние: залитая синяя иконка
   - Клик: лайк/отмена лайка
   - Анимация +1 при клике

2. **Дизлайк** 👎
   - Иконка дизлайка (без счетчика!)
   - Активное состояние: залитая иконка
   - Клик: дизлайк/отмена
   - Убирает лайк если был

3. **Поделиться** ↗️
   - Иконка "поделиться"
   - Клик открывает модальное окно с опциями
   - Счетчик поделившихся (опционально)

4. **Сохранить** 💾
   - Текст "Сохранить" + иконка
   - Клик открывает меню плейлистов
   - Добавление/удаление из плейлистов
   - Автоматический плейлист "Смотреть позже"

5. **Меню** ⋮
   - Три точки
   - Dropdown с дополнительными действиями

**Модальное окно "Поделиться":**
```
─────── Поделиться ───────
[VK] [Telegram] [WhatsApp] [Twitter]
[OK] [Viber]    [Email]    [Копировать ссылку]

Начать с: [__:__] ☐ Начать с текущего момента

Ссылка: https://platform.ru/watch?v=abc123
[Копировать]

QR-код:
[████████]
```

**Модальное окно "Сохранить в плейлист":**
```
─────── Сохранить в... ───────
☑ Смотреть позже
☐ Избранное
☐ Мой плейлист игр
☐ Обучение программированию

[+ Создать плейлист]
```

**Dropdown меню ⋮:**
- 🚨 Пожаловаться
- 💬 Открыть расшифровку (если есть)
- 📝 Показать субтитры (открыть редактор)
- ✂️ Вырезка (создать клип)
- 📊 Статистика (для владельца)

#### 3.3 Информация о канале

**Макет:**
```
[Avatar  ] Название канала              [Подписаться]
[60x60px]  1.2M подписчиков             (или [✓ Вы подписаны])
```

**Аватар канала:**
- Круглый, 60x60px (десктоп), 40x40px (мобильный)
- Кликабельный → переход на канал
- Skeleton при загрузке

**Название канала:**
- Bold, 14-16px
- Галочка верификации (если есть) ✓
- Кликабельное → переход на канал

**Счетчик подписчиков:**
- Серый цвет
- Форматирование (1.2M, 543K)
- Обновляется при подписке/отписке

**Кнопка "Подписаться":**
- Красная кнопка с белым текстом
- Hover: темнее
- После подписки: серая кнопка "Вы подписаны" с иконкой колокольчика
- Клик по "Вы подписаны" → меню:
  - 🔔 Все уведомления
  - 🔕 Персонализированные
  - 🚫 Не уведомлять
  - ❌ Отписаться

**Иконка уведомлений** (только для подписчиков):
- Колокольчик рядом с кнопкой
- Состояния:
  - 🔔 Все уведомления (залитый)
  - 🔕 Персонализированные (обычный)
  - 🚫 Отключены (перечеркнутый)

#### 3.4 Описание видео

**Свернутое состояние (по умолчанию):**
```

│ текст текст...                     │
│                                    │
│ [Ещё]                              │
└────────────────────────────────────┘
```

**Развернутое состояние:**
```
┌────────────────────────────────────┐
│ Полное описание видео              │
│ с переносами строк                 │
│                                    │
│ Ссылки кликабельные и синие        │

│                                    │
```

**Обработка ссылок:**
- Автоматическое преобразование URL в ссылки
- Поддержка timestamp ссылок (01:23)
- Упоминания каналов (@channel) кликабельны
- Хештеги (#hashtag) кликабельны
- Защита от фишинга (предупреждение о внешних ссылках)

**Временные метки (chapters):**
Если описание содержит timestamps:
```

```

Отображается как кликабельное оглавление:
- Клик перематывает видео
- Подсветка текущей главы
- Отображение в прогресс-баре плеера

### 4. Комментарии

#### 4.1 Шапка секции комментариев

```
Комментарии: 1,234     [Сортировка: ▼]
```

**Счетчик:**
- Общее количество комментариев
- Обновляется в реальном времени

**Сортировка (dropdown):**
- **Популярные** - по лайкам и ответам
- **Сначала новые** - по дате (desc)
- **Сначала старые** - по дате (asc)

#### 4.2 Форма добавления комментария

**Для авторизованных:**
```
[Avatar] [Добавить комментарий...                    ]
                                        [Отмена] [Отправить]
```

- Автофокус при клике
- Автоматическое расширение поля ввода
- Кнопки появляются при вводе текста
- Поддержка Markdown (опционально):
  - **жирный**, _курсив_, `код`
- Упоминания @username с автодополнением
- Emoji picker 😊
- Макс длина: 10,000 символов

**Для неавторизованных:**
```
[🔒] Войдите, чтобы оставить комментарий   [Войти]
```

#### 4.3 Отображение комментария

```
┌─────────────────────────────────────────┐
│ [Avatar] UserName          ❤️ 123  ⋮   │
│          2 дня назад                    │
│                                         │
│ Текст комментария с возможностью        │
│ переносов и ссылок.                     │
│                                         │
│ [Ответить] [Показать ответы (5)]       │
│                                         │
│   ├─ [Avatar] ReplyUser    ❤️ 12  ⋮   │
│   │          1 день назад               │
│   │  Ответ на комментарий               │
│   │  [Ответить]                         │
│   │                                     │
│   └─ [Показать еще 3 ответа]           │
└─────────────────────────────────────────┘
```

**Элементы комментария:**

1. **Аватар пользователя**
   - Круглый, 40x40px
   - Кликабельный → профиль

2. **Имя пользователя**
   - Bold
   - Badge автора видео (если это он)
   - Badge верификации (если есть)
   - Кликабельное → профиль

3. **Время публикации**
   - Относительное: "2 дня назад", "только что"
   - Tooltip с точной датой

4. **Текст комментария**
   - Переносы строк сохраняются
   - Ссылки кликабельны
   - @упоминания подсвечены
   - Читать далее (для длинных)

5. **Лайк кнопка**
   - Иконка сердца ❤️
   - Счетчик лайков
   - Только лайк (без дизлайка)
   - Анимация при клике

6. **Кнопка "Ответить"**
   - Открывает форму ответа под комментарием
   - Автоматическое упоминание автора

7. **Меню ⋮** (три точки)
   - **Для всех:**
     - 🚨 Пожаловаться
     - 📋 Копировать ссылку
   - **Для автора комментария:**
     - ✏️ Изменить
     - 🗑 Удалить
   - **Для владельца видео:**
     - 📌 Закрепить (топ комментарий)
     - ❤️ Отметить сердцем (от создателя)
     - 🚫 Скрыть пользователя

8. **Ответы (replies)**
   - Отступ слева
   - Максимум 1 уровень вложенности
   - Свернуты по умолчанию
   - "Показать ответы (N)" для раскрытия
   - "Скрыть ответы" для сворачивания

## Технические детали

### API endpoints

```typescript
// Получение данных видео
GET /api/v1/videos/{id}
Response: {
  video: VideoDetails,
  channel: ChannelInfo,
  related: Video[]
}

// Лайк/дизлайк
POST /api/v1/videos/{id}/like
POST /api/v1/videos/{id}/dislike
DELETE /api/v1/videos/{id}/like

// Подписка на канал
POST /api/v1/channels/{id}/subscribe
DELETE /api/v1/channels/{id}/subscribe

// Получение комментариев
GET /api/v1/videos/{id}/comments
Query: sort=popular|newest|oldest, limit=20, offset=0
Response: {
  comments: Comment[],
  total: number,
  hasMore: boolean
}

// Создание комментария
POST /api/v1/videos/{id}/comments
Body: { text: string, parent_id?: string }

// Обновление/удаление комментария
PATCH /api/v1/comments/{id}
DELETE /api/v1/comments/{id}

// Лайк комментария
POST /api/v1/comments/{id}/like
DELETE /api/v1/comments/{id}/like

// Сохранение прогресса
POST /api/v1/videos/{id}/progress
Body: { time: number }

// Отчет о просмотре (аналитика)
POST /api/v1/videos/{id}/view
Body: { duration: number, quality: string }
```

### Типы данных

```typescript
interface VideoDetails extends Video {
  description: string;
  tags: string[];
  category: string;
  chapters?: Chapter[];
  transcript?: TranscriptLine[];
  is_liked: boolean;
  is_disliked: boolean;
  is_saved: boolean;
  watch_progress?: number; // секунды
}

interface Comment {
  id: string;
  author: {
    id: string;
    username: string;
    avatar: string;
    is_verified: boolean;
    is_video_author: boolean;
  };
  text: string;
  likes: number;
  is_liked: boolean;
  created_at: string;
  edited_at?: string;
  replies_count: number;
  replies?: Comment[];
  is_pinned: boolean;
  is_hearted: boolean;
  parent_id?: string;
}
```
