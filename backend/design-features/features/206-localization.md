# 206 - Локализация и мультиязычность

**Приоритет:** Средний (206)
**Категория:** Интернационализация

## Поддерживаемые языки
- Русский (основной)
- Английский
- Украинский
- Казахский
- Беларусский
- (расширяемо)

## Локализация интерфейса
- Переводы всех UI элементов
- Форматы дат/времени
- Форматы чисел
- Валюты

## Локализация контента
- Метаданные видео на разных языках
- Переводы названий и описаний
- Субтитры на разных языках
- Автопереводы (опционально)

## Определение языка
- По настройкам браузера
- По IP геолокации
- Ручной выбор пользователя

## API
```typescript
GET /api/v1/locales
GET /api/v1/translations/{locale}
```
