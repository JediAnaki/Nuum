# 303 - Stories (Истории)

**Приоритет:** Низкий (303)
**Категория:** Контент

## Описание
Временный контент (24 часа) для быстрой коммуникации с аудиторией.

## Функции
- Фото и видео до 15 сек
- Текст, стикеры, эмодзи
- Опросы и вопросы
- Ссылки (для верифицированных)

## Просмотр
- Горизонтальная прокрутка в шапке
- Tap для следующей сторис
- Автопроигрывание

## Аналитика
- Просмотры
- Выходы
- Ответы
