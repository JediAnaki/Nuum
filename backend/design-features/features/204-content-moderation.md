# 204 - Модерация контента

**Приоритет:** Средний (204)
**Категория:** Безопасность и соответствие
**Зависимости:** 005 (Загрузка), 004 (Комментарии)

## Описание
Система модерации для соблюдения правил сообщества и законодательства.

## Автоматическая модерация

### Видео
- Проверка на авторские права (Content ID)
- Распознавание запрещенного контента (CV)
- Детекция насилия, наготы
- Проверка возрастных ограничений
- Спам-детектор

### Комментарии
- Фильтр матерных слов
- Спам-фильтр
- Ссылки и фишинг
- Hate speech detection

## Ручная модерация

### Жалобы
- Система репортов
- Очередь на проверку
- Приоритизация по серьезности

### Действия
- Предупреждение
- Удаление контента
- Демонетизация
- Strike (страйк)
- Блокировка канала

## Правила сообщества
- Запрет насилия
- Запрет hate speech
- Авторские права
- Возрастные ограничения
- Дезинформация

## Апелляции
- Обжалование решений
- Ручная проверка
- Восстановление контента
