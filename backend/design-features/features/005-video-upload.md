# 005 - Загрузка и публикация видео

**Приоритет:** MVP (005)
**Категория:** Создание контента
**Зависимости:** 001 (Авторизация)

## Описание

Полноценная система загрузки, обработки и публикации видео. Позволяет авторам загружать контент, настраивать метаданные, превью и параметры публикации.

## Функциональные требования

### 1. Точки входа для загрузки

#### 1.1 Кнопка в шапке
```
[+ Создать]
```
- Dropdown меню:
  - 📹 Загрузить видео
  - 🔴 Начать прямой эфир
  - 📝 Создать пост (будущая фича)

#### 1.2 Прямой URL
```
/studio/upload
/upload
```

#### 1.3 Перетаскивание (Drag & Drop)
- На любую страницу платформы (для авторизованных)
- Появление оверлея:
```
┌─────────────────────────────────────┐
│                                     │
│         📹                          │
│   Перетащите видео сюда             │
│   для загрузки                      │
│                                     │
└─────────────────────────────────────┘
```

### 2. Процесс загрузки

#### 2.1 Шаг 1: Выбор файла

**Страница выбора:**
```
┌─────────────────────────────────────┐
│     Загрузка видео                  │
│                                     │
│          [📹]                       │
│                                     │
│   Перетащите файлы сюда             │
│          или                        │
│   [Выбрать файлы]                   │
│                                     │
│   Поддерживаемые форматы:           │
│   MP4, MOV, AVI, WMV, FLV, MKV     │
│                                     │
│   Макс. размер: 10 ГБ               │
│   Макс. длительность: 12 часов      │
└─────────────────────────────────────┘
```

**Поддерживаемые форматы:**
- Видео: MP4 (H.264, H.265), MOV, AVI, WMV, FLV, MKV, WebM
- Аудио: AAC, MP3, WAV (будет конвертировано)
- Макс. размер: 10 ГБ (128 ГБ для верифицированных)
- Макс. длительность: 12 часов

**Валидация:**
- Проверка формата файла
- Проверка размера
- Проверка разрешения (мин. 426x240)
- Проверка кодека
- Проверка битрейта

**Ошибки:**
- "Неподдерживаемый формат"
- "Файл слишком большой"
- "Видео слишком длинное"
- "Низкое качество видео (мин. 240p)"

#### 2.2 Шаг 2: Загрузка файла

**Индикатор загрузки:**
```
┌─────────────────────────────────────┐
│ video_name.mp4              [X]     │
│                                     │
│ Загрузка...                         │
│ ████████████░░░░░░░░ 65%            │
│                                     │
│ 650 МБ из 1 ГБ                     │
│ Осталось: ~3 минуты                 │
│ Скорость: 5 МБ/с                    │
│                                     │
│ [Пауза]  [Отмена]                  │
└─────────────────────────────────────┘
```

**Функции:**
- Resumable upload (можно приостановить и продолжить)
- Chunked upload (загрузка частями по 5 МБ)
- Автовозобновление при потере соединения
- Загрузка в фоне (можно переключать вкладки)
- Прогресс сохраняется между сессиями

**Параллельная загрузка:**
- До 5 файлов одновременно
- Список файлов с отдельными прогресс-барами
- Общий прогресс загрузки

**После загрузки:**
```
✓ Загружено успешно!
  Обработка видео...

[Продолжить к деталям]
```


**Валидация полей:**

1. **Название** (обязательно)
   - Мин. 3 символа
   - Макс. 100 символов
   - Не должно содержать только знаки препинания
   - Проверка на спам-слова

2. **Описание**
   - Макс. 5000 символов
   - Поддержка переносов строк
   - Автолинкование URLs
   - Поддержка timestamps (00:00)

3. **Превью** (обязательно)
   - Минимум одно превью
   - Формат: JPG, PNG, WebP
   - Разрешение: 1280x720 (16:9)
   - Размер: до 2 МБ
   - Можно загрузить свое или выбрать из автогенерированных

4. **Теги**
   - До 15 тегов
   - Длина тега: 2-30 символов
   - Автодополнение популярных тегов
   - Запрет спам-тегов

5. **Категория**
   - Обязательна для выбора
   - Список: Игры, Музыка, Новости, Обучение, Спорт, и т.д.

### 3. Управление превью

#### 3.1 Автогенерация
- 3 кадра из видео (начало, середина, конец)
- Автоматически при загрузке
- Отображаются как варианты выбора

#### 3.2 Загрузка своего превью
```
[+ Загрузить превью]
→ File picker
→ Валидация (16:9, мин. 1280x720)
→ Crop tool если не 16:9
→ Предпросмотр
→ Сохранение
```

**Crop tool:**
```
┌─────────────────────────────────────┐
│ Обрезка превью                 [X]  │
├─────────────────────────────────────┤
│                                     │
│  ┌───────────────────────┐          │
│  │  ┏━━━━━━━━━━━━━━━━┓  │          │
│  │  ┃                  ┃  │          │
│  │  ┃   Crop area      ┃  │          │
│  │  ┃   (16:9)         ┃  │          │
│  │  ┗━━━━━━━━━━━━━━━━┛  │          │
│  └───────────────────────┘          │
│                                     │
│  [Отмена]           [Применить]    │
└─────────────────────────────────────┘
```

#### 3.3 Редактор превью (расширенный)
- Фильтры (яркость, контраст, насыщенность)
- Наложение текста
- Наложение стикеров/эмодзи
- Шаблоны дизайна

### 7. Обработка видео

#### 7.1 Этапы обработки

```
Обработка видео...
┌─────────────────────────────────────┐
│ ✓ Загрузка завершена                │
│ ✓ Проверка файла                    │
│ ✓ Извлечение метаданных             │
│ ⏳ Транскодирование (45%)           │
│   • SD (480p) ✓                     │
│   • HD (720p) ✓                     │
│   • Full HD (1080p) ⏳              │
│   • 4K (2160p) - в очереди          │
│ ⏳ Генерация превью                 │
│ ⏳ Создание субтитров               │
│ ⏱ Публикация (ожидание)             │
│                                     │
│ Примерное время: ~15 минут          │
└─────────────────────────────────────┘
```

**Процессы:**
1. **Валидация** - проверка формата, кодека
2. **Транскодирование** - конвертация в разные качества
3. **Превью** - генерация thumbnail и превью-кадров
4. **Субтитры** - автогенерация (если включено)
5. **Индексация** - добавление в поиск и рекомендации

#### 7.2 Параллельная публикация

- Можно опубликовать до завершения обработки всех качеств
- Сначала доступны низкие качества (480p)
- Высокие качества (1080p, 4K) добавляются по мере готовности
- Уведомление подписчикам после обработки минимум HD (720p)

## UI/UX требования

### Адаптивность

#### Мобильная версия
- Упрощенная форма (только основные поля)
- Вертикальный стек элементов
- Продвинутые настройки в отдельном экране
- Превью меньшего размера
- Crop tool адаптирован под касания

#### Десктоп
- Две колонки (форма + превью)
- Все настройки доступны
- Расширенные инструменты редактирования

### Сохранение прогресса

**Автосохранение:**
- Каждые 30 секунд
- При потере фокуса
- При переходе между шагами
- Сохранение в localStorage + backend

**Восстановление:**
```
┌────────────────────────────────────┐
│ У вас есть незавершенная загрузка  │
│                                    │
│ video_name.mp4                     │
│ Загружено: 65%                     │
│                                    │
│ [Удалить]     [Продолжить]        │
└────────────────────────────────────┘
```

## Технические детали

### API endpoints

```typescript
// Инициация загрузки
POST /api/v1/videos/upload/init
Body: {
  filename: string,
  size: number,
  mime_type: string
}
Response: {
  upload_id: string,
  chunk_size: number,
  upload_url: string
}

// Загрузка чанка
PUT /api/v1/videos/upload/{upload_id}/chunk/{chunk_number}
Body: <binary data>
Headers: Content-Range, Content-Type

// Завершение загрузки
```

### Chunked upload

```javascript
// Библиотека: tus-js-client (resumable uploads)
import * as tus from "tus-js-client";

const upload = new tus.Upload(file, {
  endpoint: "/api/v1/videos/upload",
  retryDelays: [0, 3000, 5000, 10000],
  chunkSize: 5 * 1024 * 1024, // 5 MB
  metadata: {
    filename: file.name,
    filetype: file.type,
  },
  onProgress: (bytesUploaded, bytesTotal) => {
    const percentage = ((bytesUploaded / bytesTotal) * 100).toFixed(2);
    console.log(percentage + "%");
  },
  onSuccess: () => {
    console.log("Upload finished");
  },
});

upload.start();
```
