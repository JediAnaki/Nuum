# 007 - Профиль пользователя

**Приоритет:** MVP (007)
**Категория:** Пользователи и персонализация
**Зависимости:** 001 (Авторизация)

## Описание

Личный профиль зарегистрированного пользователя с возможностью управления аккаунтом, настройками, библиотекой и активностью.

## Функциональные требования

### 1. Страница профиля

**URL:** `/user/@username` или `/user/{user_id}`

**Структура:**
```
┌─────────────────────────────────────────┐
│ [Аватар   ] @username                   │
│ [120x120  ] Полное имя                  │
│             Дата регистрации: июнь 2024 │
│             [Редактировать профиль]     │
├─────────────────────────────────────────┤
│ [Активность] [Плейлисты] [О себе]       │
├─────────────────────────────────────────┤
│ Последняя активность:                   │
│ • Поставил лайк видео "..."             │
│ • Подписался на канал "..."             │
│ • Добавил в плейлист "..."              │
└─────────────────────────────────────────┘
```

**Вкладки:**

1. **Активность** - публичные действия (лайки, комментарии)
2. **Плейлисты** - публичные плейлисты пользователя
3. **О себе** - биография, ссылки, статистика

### 2. Настройки аккаунта

**URL:** `/settings/account`

**Секции:**

#### 2.1 Общие настройки
- **Аватар** - загрузка/изменение (макс. 2 МБ, 800x800px)
- **Имя пользователя** (@username) - уникальное, 3-30 символов
- **Полное имя** - отображаемое имя
- **Email** - с подтверждением при изменении
- **Биография** - до 500 символов
- **Ссылки** - до 5 внешних ссылок

#### 2.2 Приватность
- **Профиль** - публичный/приватный
- **Подписки** - видимость списка подписок
- **Плейлисты** - по умолчанию публичные/приватные
- **Активность** - показывать лайки и комментарии
- **История просмотров** - сохранять/не сохранять

#### 2.3 Безопасность
- **Изменение пароля**
- **Двухфакторная аутентификация** (2FA)
- **Активные сессии** - управление устройствами
- **Связанные аккаунты** - OAuth провайдеры
- **Журнал входов** - последние 50 входов

#### 2.4 Уведомления
- **Email уведомления:**
  - Новое видео от подписок
  - Ответы на комментарии
  - Упоминания
  - Новости платформы
- **Push уведомления** (браузерные)
- **Частота** - мгновенно/дайджест/никогда

### 3. Библиотека

**URL:** `/library`

**Секции:**

```
┌─────────────────────────────────────────┐
│ [История] [Смотреть позже] [Плейлисты]  │
│ [Понравилось] [Покупки]                 │
├─────────────────────────────────────────┤
│ История просмотров                      │
│ ┌─────────────────┐                     │
│ │ Сегодня         │ [Очистить всю]      │
│ │ [Видео] [Видео] │ [Приостановить]     │
│ │ Вчера           │                     │
│ │ [Видео] [Видео] │                     │
│ └─────────────────┘                     │
└─────────────────────────────────────────┘
```

#### 3.1 История просмотров
- Группировка по дням
- Поиск по истории
- Удаление отдельных видео
- Очистка всей истории
- Приостановка истории
- Автоудаление старше X дней (настройка)

#### 3.2 Смотреть позже
- Сохраненные видео для просмотра
- Автоматический плейлист
- Порядок добавления/произвольный
- Отметка просмотренных
- Удаление просмотренных

#### 3.3 Понравилось
- Все лайкнутые видео
- Сортировка по дате
- Фильтр по категориям
- Публичный/приватный плейлист

#### 3.4 Мои плейлисты
- Созданные плейлисты
- Сохраненные плейлисты (от других)
- Создание нового плейлиста
- Редактирование/удаление

### 4. Управление плейлистами

#### 4.1 Создание плейлиста

**Форма:**
```
Новый плейлист
- Название (макс. 100 символов)
- Описание (макс. 500 символов)
- Видимость: Публичный/По ссылке/Приватный
- Добавить видео (поиск или ID)
```

#### 4.2 Редактирование плейлиста
- Изменение метаданных
- Добавление/удаление видео
- Изменение порядка (drag & drop)
- Дублирование плейлиста
- Удаление плейлиста

### 5. Подписки

**URL:** `/subscriptions`

**Управление:**
```
┌─────────────────────────────────────────┐
│ Мои подписки (123)         [Управление] │
├─────────────────────────────────────────┤
│ [Avatar] Название канала    [✓ Подписан]│
│          1.2M подписчиков   [🔔 Все]    │
│                                         │
│ [Avatar] Название канала 2  [✓ Подписан]│
│          543K подписчиков   [🔕 Персон.] │
└─────────────────────────────────────────┘
```

**Функции:**
- Список всех подписок
- Сортировка (по имени, дате подписки, активности)
- Группировка по категориям (опционально)
- Настройка уведомлений для каждого канала
- Массовая отписка
- Экспорт списка подписок (CSV)

### 6. Статистика пользователя

**Приватная статистика (только для владельца):**
- Всего просмотрено видео
- Общее время просмотра
- Среднее время в день
- Любимые категории (топ-5)
- Любимые каналы (топ-10)
- График активности по дням недели
- График просмотров по часам

## UI/UX требования

### Адаптивность
- **Мобильный:** вертикальная навигация, упрощенные формы
- **Планшет:** двухколоночный layout
- **Десктоп:** полная функциональность, sidebar навигация

### Валидация
- Real-time проверка уникальности username
- Проверка форматов (email, URL)
- Индикация силы пароля
- Предупреждения перед удалением

### Уведомления
- Toast сообщения об успехе/ошибке
- Подтверждение критичных действий
- Сохранение прогресса форм

## Технические детали

### API endpoints
```typescript
// Профиль
GET /api/v1/users/{id}
PATCH /api/v1/users/{id}
DELETE /api/v1/users/{id}

// Настройки
GET /api/v1/users/me/settings
PATCH /api/v1/users/me/settings

// Библиотека
GET /api/v1/users/me/history
DELETE /api/v1/users/me/history/{video_id}
DELETE /api/v1/users/me/history

GET /api/v1/users/me/watch-later
POST /api/v1/users/me/watch-later/{video_id}
DELETE /api/v1/users/me/watch-later/{video_id}

GET /api/v1/users/me/liked
GET /api/v1/users/me/subscriptions
GET /api/v1/users/me/playlists

// Статистика
GET /api/v1/users/me/stats
```

### Типы данных
```typescript
interface UserProfile {
  id: string;
  username: string;
  full_name?: string;
  email: string;
  avatar?: string;
  bio?: string;
  links: string[];
  created_at: string;
  is_verified: boolean;
  stats: {
    videos_watched: number;
    total_watch_time: number;
    playlists: number;
    subscriptions: number;
  };
  privacy: {
    show_subscriptions: boolean;
    show_playlists: boolean;
    show_activity: boolean;
  };
}

interface UserSettings {
  notifications: NotificationSettings;
  privacy: PrivacySettings;
  playback: PlaybackSettings;
}
```

## Метрики успеха
- Completion rate редактирования профиля > 70%
- Использование библиотеки > 60% пользователей
- Средний размер истории > 50 видео
- Активных плейлистов на пользователя > 3

## Особенности для русскоязычной аудитории
- Поддержка кириллицы в username
- Локализованные даты и времена
- Русские форматы чисел
- Экспорт данных в соответствии с GDPR/российским законодательством
