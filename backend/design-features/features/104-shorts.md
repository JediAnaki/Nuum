# 104 - Shorts (короткие вертикальные видео)

**Приоритет:** Высокий (104)
**Категория:** Контент
**Зависимости:** 002 (Плеер), 003 (Главная)

## Описание
Короткие вертикальные видео до 60 секунд, аналог TikTok/Reels.

## URL
`/shorts/{video_id}`

## Характеристики
- **Формат:** 9:16 (вертикальный)
- **Длительность:** до 60 секунд
- **Качество:** до 1080x1920

## Плеер Shorts

### Интерфейс (mobile-first)
```
┌─────────────────┐
│                 │ ← Видео на весь экран
│                 │
│                 │
│  [Информация]   │ ← Справа внизу
│  [@Канал]       │
│  [Описание]     │

│                 │
│  [👍 12K]       │ ← Правая панель
│  [👎]           │   действий
│  [💬 234]       │
│  [↗️ 45]        │
│  [⋮]            │
│                 │
│ [◀ Пред | След ▶]│ ← Навигация
└─────────────────┘
```

### Навигация
- **Свайп вверх** - следующее Shorts
- **Свайп вниз** - предыдущее Shorts
- **Tap** - пауза/воспроизведение
- **Двойной tap** - лайк

### Лента Shorts
`/shorts`
- Бесконечная вертикальная прокрутка
- Автопроигрывание при появлении в viewport
- Алгоритм рекомендаций

## Создание Shorts

### Загрузка
- Через `/upload` с выбором "Shorts"
- Автоопределение по соотношению сторон и длине
- Обрезка длинного видео до 60 сек

### Редактор (будущая фича)
- Обрезка
- Музыка из библиотеки
- Текст и стикеры
- Эффекты

## API
```typescript
GET /api/v1/shorts
GET /api/v1/shorts/{id}
POST /api/v1/shorts
```

## Метрики
- Среднее Shorts на сессию > 10
- Completion rate > 70%
- Share rate > 15%
