# 207 - Доступность (Accessibility)

**Приоритет:** Средний (207)
**Категория:** UX и инклюзивность

## WCAG 2.1 соответствие (уровень AA)

### Визуальная доступность
- Достаточный контраст текста (4.5:1)
- Масштабирование текста до 200%
- Режим высокой контрастности
- Темная/светлая темы

### Клавиатурная навигация
- Tab navigation по всем элементам
- Skip links
- Focus indicators
- Горячие клавиши

### Экранные читалки
- ARIA атрибуты
- Семантический HTML
- Alt текст для изображений
- Описания для видео

### Субтитры и транскрипты
- Автоматические субтитры
- Пользовательские субтитры
- Транскрипты видео
- Аудио описания

### Настройки
- Отключение анимаций (prefers-reduced-motion)
- Размер текста
- Цветовые схемы
- Скорость воспроизведения

## Тестирование
- Lighthouse accessibility audit
- axe DevTools
- Ручное тестирование
