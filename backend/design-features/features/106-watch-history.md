# 106 - История просмотров

**Приоритет:** Высокий (106)
**Категория:** Персонализация
**Зависимости:** 001 (Авторизация), 007 (Профиль)

## Описание
Полная история просмотренных видео с возможностью управления и поиска.

## URL
`/feed/history`

## Функции

### Просмотр истории
```
┌─────────────────────────────────┐
│ История просмотров              │
│ [Поиск: _______] [Очистить всю] │
├─────────────────────────────────┤
│ Сегодня                         │
│ [Видео 1] Просмотрено: 14:30    │
│ [Видео 2] Просмотрено: 12:15    │
│                                 │
│ Вчера                           │
│ [Видео 3] Просмотрено: 22:45    │
│ ...                             │
└─────────────────────────────────┘
```

### Группировка
- По дням
- По неделям
- По месяцам

### Поиск
- По названию видео
- По каналу
- По дате

### Управление
- Удалить отдельное видео
- Удалить всю историю
- Удалить за период
- Приостановить историю
- Автоудаление старше X дней

### Продолжить просмотр
- Маркер прогресса просмотра
- Автоматическое возобновление с места остановки
- Синхронизация между устройствами

## Настройки

### Приватность
- Сохранять историю (вкл/выкл)
- Включить в рекомендации
- Показывать в активности профиля
- Автоочистка

## API
```typescript
GET /api/v1/users/me/history
POST /api/v1/users/me/history (добавить просмотр)
DELETE /api/v1/users/me/history/{video_id}
DELETE /api/v1/users/me/history

PATCH /api/v1/users/me/history/pause
```

## Метрики
- Использование истории > 50%
- Продолжение просмотра > 30%
- Средний размер истории > 100 видео
