# 103 - Лента подписок

**Приоритет:** Высокий (103)
**Категория:** Контент и персонализация
**Зависимости:** 001 (Авторизация), 008 (Каналы)

## Описание
Персонализированная лента новых видео от каналов на которые подписан пользователь.

## URL
`/feed/subscriptions`

## Функции

### Сортировка
- **Новые** (по дате публикации)
- **Все** (включая старые непросмотренные)

### Фильтры
- По каналам (выбор из списка)
- Только непросмотренные
- Живые трансляции
- Shorts

### Группировка
- Сегодня
- Вчера
- Эта неделя
- Раньше

### Уведомления
- Badge с количеством новых видео
- Уведомления по настройке канала (все/персонализированные/нет)

## Особенности
- Показываются ВСЕ видео от подписок (не алгоритм)
- Опция "Отметить все просмотренными"
- Фильтр "Только каналы с включенными уведомлениями"

## API
```typescript
GET /api/v1/feed/subscriptions
Query: sort?, filter?, channel_ids?
```

## Метрики
- Посещаемость > 70% подписанных
- Среднее видео на посещение > 3
- CTR > 25%
