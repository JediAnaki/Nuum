# 109 - Категории контента

**Приоритет:** Высокий (109)
**Категория:** Навигация и организация
**Зависимости:** 003 (Главная)

## Описание
Система категоризации видео для удобной навигации и обнаружения контента.

## Категории

### Основные категории:
1. 🎮 **Игры** - геймплей, летсплеи, обзоры
2. 🎵 **Музыка** - клипы, концерты, каверы
3. 🎓 **Образование** - уроки, лекции, туториалы
4. ⚽ **Спорт** - матчи, highlights, обзоры
5. 📰 **Новости и политика**
6. 🎬 **Фильмы и ТВ** - трейлеры, рецензии
7. 🎭 **Развлечения** - комедия, влоги, челленджи
8. 💄 **Красота и мода** - макияж, стиль, обзоры
9. 🍳 **Еда и кулинария** - рецепты, обзоры
10. 🛠 **Сделай сам** - DIY, ремонт, лайфхаки
11. 🚗 **Авто и транспорт**
12. 🐾 **Животные** - питомцы, дикая природа
13. 💼 **Бизнес** - предпринимательство, финансы
14. 🎨 **Творчество** - рисование, музыка, ремесла
15. 🏠 **Образ жизни** - путешествия, влоги
16. 🎤 **Люди и блоги**
17. 🔬 **Наука и технологии**

## Страница категории

### URL
`/category/{slug}` (например `/category/gaming`)

### Структура
```
🎮 Игры

[ Популярное ] [ Новое ] [ Живое ]

[Видео сетка с фильтрами]
```

### Подкатегории
- Каждая категория может иметь подкатегории
- Фильтр по подкатегориям

## Технические детали

### Иерархия
```typescript
interface Category {
  id: string;
  slug: string;
  name: string;
  icon: string;
  parent_id?: string;
  subcategories?: Category[];
  video_count: number;
}
```

### API
```typescript
GET /api/v1/categories
GET /api/v1/categories/{slug}/videos
```
