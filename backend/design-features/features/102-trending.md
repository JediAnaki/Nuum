# 102 - Популярное (Trending)

**Приоритет:** Высокий (102)
**Категория:** Обнаружение контента
**Зависимости:** 003 (Главная)

## Описание
Страница с трендовыми видео, набирающими популярность прямо сейчас.

## URL
`/trending`

## Основные разделы

### Вкладки категорий
```
[ Все ] [ Музыка ] [ Игры ] [ Новости ] [ Спорт ]
```

### Алгоритм трендов
- Резкий рост просмотров (за последние 24-48ч)
- Высокий engagement (лайки, комментарии, шары)
- Свежесть (не старше 7 дней)
- Разнообразие каналов
- Региональность

### Обновление
- Каждые 15 минут
- Индикатор "Обновлено X минут назад"

## API
```typescript
GET /api/v1/trending
Query: category?, region?
```

## Метрики
- CTR на trending > 20%
- Разнообразие каналов > 50 в топ-50
