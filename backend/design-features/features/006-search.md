# 006 - Поиск

**Приоритет:** MVP (006)
**Категория:** Навигация и обнаружение контента
**Зависимости:** 003 (Главная страница)

## Описание

Полнофункциональная система поиска видео, каналов и плейлистов с фильтрами, сортировкой и автодополнением. Ключевой инструмент для обнаружения контента.

## Функциональные требования

### 1. Поисковая строка

#### 1.1 Расположение
```
┌──────────────────────────────────────────┐
│ [☰] [Logo]  [🔍 Поиск...        ] [🎤] │
└──────────────────────────────────────────┘
```

**Элементы:**
- Иконка лупы слева
- Поле ввода с placeholder "Поиск..."
- Кнопка голосового поиска справа 🎤
- Кнопка поиска (Enter или клик по лупе)

#### 1.2 Состояния

**Не в фокусе:**
```
[🔍 Поиск...                             ]
```

**В фокусе:**
```
[🔍 майнкрафт выживание                 ][X]
 ┌────────────────────────────────────────┐
 │ 🕐 майнкрафт выживание                 │
 │ 🕐 майнкрафт строительство             │
 │ 🔍 майнкрафт выживание хардкор         │
 │ 🔍 майнкрафт выживание с модами        │
 │ 🔍 майнкрафт выживание 2024            │
 └────────────────────────────────────────┘
```

**С текстом:**
- Кнопка очистки [X] появляется
- Автодополнение в реальном времени
- История поиска (иконка 🕐)
- Популярные запросы (иконка 🔍)

### 2. Автодополнение

#### 2.1 Источники подсказок

1. **История поиска** (топ 5)
   - Личные последние запросы
   - Иконка 🕐
   - Кнопка удаления из истории [X]

2. **Популярные запросы** (топ 10)
   - На основе текущего ввода
   - Иконка 🔍
   - Глобальная статистика

3. **Каналы** (топ 3)
   - [Avatar] Название канала
   - Количество подписчиков
   - Badge верификации

4. **Видео** (топ 3, если точное совпадение)
   - [Thumbnail] Название видео
   - Просмотры • дата

#### 2.2 Поведение

**Навигация клавиатурой:**
- ↓ / ↑ - перемещение по подсказкам
- Enter - выбор подсказки
- Escape - закрыть подсказки
- Tab - автодополнение первой подсказкой

**Клик мышью:**
- Клик по подсказке → поиск
- Hover → подсветка
- Клик вне → закрыть

**Дебаунс:**
- Задержка 300ms перед запросом
- Отмена предыдущего запроса при новом вводе

**Живое:**

- Сортировка по зрителям

## Технические детали

### API endpoints

```typescript
// Основной поиск
GET /api/v1/search
Query params:
  - q: string (запрос)
  - type?: "video" | "channel" | "playlist" | "live"
  - upload_date?: "hour" | "today" | "week" | "month" | "year"
  - duration?: "short" | "medium" | "long"
  - features?: string[] (hd, 4k, subtitles, etc.)
  - sort?: "relevance" | "date" | "views" | "rating"
  - page?: number
  - limit?: number (default: 20, max: 50)

Response: {
  results: SearchResult[],
  total: number,
  page: number,
  hasMore: boolean,
  corrected_query?: string,
  suggestions?: string[]
}

// Автодополнение
GET /api/v1/search/autocomplete
Query: q=string
Response: {
  suggestions: string[],
  channels: Channel[],
  videos?: Video[]
}

// История поиска
GET /api/v1/search/history
POST /api/v1/search/history (добавить)
DELETE /api/v1/search/history/{id}
DELETE /api/v1/search/history (очистить всё)

// Голосовой поиск (если backend обработка)
POST /api/v1/search/voice
Body: { audio: base64, language: "ru" }
Response: { text: string }
```

### Типы данных

```typescript
type SearchResult = VideoResult | ChannelResult | PlaylistResult;

interface VideoResult {
  type: "video";
  video: Video;
  relevance_score: number;
  matched_fields: ("title" | "description" | "tags")[];
}

interface ChannelResult {
  type: "channel";
  channel: Channel;
  relevance_score: number;
  latest_videos: Video[];
}

interface PlaylistResult {
  type: "playlist";
  playlist: Playlist;
  relevance_score: number;
}
```

### Поисковый движок

**Elasticsearch / OpenSearch:**
```json
{
  "query": {
    "multi_match": {
      "query": "майнкрафт выживание",
      "fields": [
        "title^3",
        "description^2",
        "tags^2",
        "channel_name",
        "transcript"
      ],
      "type": "best_fields",
      "fuzziness": "AUTO"
    }
  },
  "filter": {
    "range": {
      "published_at": { "gte": "now-7d" }
    }
  },
  "sort": [
    { "_score": "desc" },
    { "views": "desc" }
  ]
}
```

**Функции:**
- Fuzzy matching (опечатки)
- Морфология (стемминг русского)
- Синонимы
- Стоп-слова
- N-граммы для автодополнения

### Индексация

**Данные для индекса:**
```typescript
interface SearchDocument {
  id: string;
  type: "video" | "channel" | "playlist";
  title: string;
  description: string;
  tags: string[];
  category: string;
  language: string;
  channel_name: string;
  published_at: Date;
  views: number;
  likes: number;
  engagement_score: number; // views + likes + comments
  duration?: number;
  is_live: boolean;
  features: string[]; // hd, 4k, subtitles, etc.
  transcript?: string;
}
```

**Обновление индекса:**
- Real-time при публикации видео
- Batch обновление статистики (каждые 5 мин)
- Ребилд индекса раз в день (оффпик)

### Кэширование

### Производительность

**Оптимизации:**
- Дебаунс автодополнения (300ms)
- Отмена предыдущих запросов
- Prefetch следующей страницы
- CDN для статики
- Lazy loading результатов

**Метрики:**
