# 305 - Совместный просмотр (Watch Parties)

**Приоритет:** Низкий (305)
**Категория:** Социальные функции

## Описание
Синхронизированный просмотр видео с друзьями.

## Функции
- Создание комнаты просмотра
- Приглашение по ссылке
- Синхронное воспроизведение
- Общий чат
- Реакции в реальном времени

## Управление
- Хост контролирует плеер
- Передача управления
- Голосование за следующее видео
