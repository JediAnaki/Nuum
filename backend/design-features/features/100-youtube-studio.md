# 100 - YouTube Studio (панель создателя)

**Приоритет:** Высокий (100)
**Категория:** Создание контента и аналитика
**Зависимости:** 001 (Авторизация), 005 (Загрузка видео), 008 (Канал)

## Описание

Панель управления для авторов контента с аналитикой, управлением видео, комментариями и монетизацией.

## Основные разделы

### 1. URL
`/studio` - главная страница Studio

### 2. Навигация Studio

**Sidebar:**
- 📊 Главная (обзор)
- 📹 Контент (видео, Shorts, live, посты)
- 💬 Комментарии
- 📈 Аналитика
- 💰 Монетизация
- ⚙️ Настройки

- Аналитика по видео

### 5. Аналитика

#### 5.1 Обзор
- Просмотры (график за период)
- Время просмотра
- Прирост подписчиков
- Доход (если монетизация)

#### 5.2 Контент
- Топ видео по просмотрам
- Топ по времени просмотра
- Источники трафика
- Демография зрителей

#### 5.3 Аудитория
- Уникальные зрители
- Возвращаемость зрителей
- Когда зрители онлайн
- Где смотрят (география)
- Возраст и пол

#### 5.4 Детальная аналитика видео

**Метрики:**
- Просмотры по времени (график)
- Средняя длительность просмотра
- % просмотренного
- Heat map (где зрители уходят)
- Удержание аудитории
- Клики по экранам и карточкам
- Источники трафика
- Ключевые слова поиска

**Сравнение:**
- Сравнение с предыдущими видео
- Сравнение с аналогичными каналами

### 6. Комментарии

**Фильтры:**
- Опубликованные
- Ожидают проверки
- Возможный спам

**Действия:**
- Одобрить/отклонить
- Ответить
- Удалить
- Заблокировать пользователя
- Отметить сердцем
- Закрепить

**Массовые действия:**
- Одобрить выбранные
- Удалить выбранные

### 7. Настройки канала

#### 7.1 Общие
- Название канала
- Описание
- Email для связи
- Ссылки

#### 7.2 Брендинг
- Аватар
- Баннер
- Водяной знак для видео

#### 7.3 Загрузки по умолчанию
- Видимость
- Категория
- Язык
- Лицензия
- Разрешения (комментарии, embedding)

#### 7.4 Разрешения
- Управление доступом к каналу
- Добавление менеджеров
- Роли и права доступа

#### 7.5 Сообщество
- Настройка вкладки "Сообщество"
- Модерация комментариев
- Автоматические фильтры
- Черный список слов

## Технические детали

### API endpoints
```typescript
// Studio API
GET /api/v1/studio/dashboard
GET /api/v1/studio/videos
PATCH /api/v1/studio/videos/{id}
DELETE /api/v1/studio/videos/{id}

GET /api/v1/studio/analytics
GET /api/v1/studio/analytics/videos/{id}

GET /api/v1/studio/comments
PATCH /api/v1/studio/comments/{id}

GET /api/v1/studio/revenue (если монетизация)
```

### Типы данных
```typescript
interface StudioDashboard {
  channel: {
    id: string;
    name: string;
    subscribers: number;
  };
  stats: {
    views: number;
    watch_time: number;
    subscribers_gained: number;
    revenue?: number;
  };
  period: {
    start: string;
    end: string;
  };
  comparison: {
    views_change: number; // процент
    watch_time_change: number;
  };
}

interface VideoAnalytics {
  video_id: string;
  views: number;
  watch_time: number;
  avg_view_duration: number;
  avg_view_percentage: number;
  likes: number;
  dislikes: number;
  comments: number;
  shares: number;
  traffic_sources: {
    source: string;
    views: number;
  }[];
  retention: {
    time: number;
    percentage: number;
  }[];
}
```

## UI/UX требования

- **Дашборд стиль** - светлая тема (опционально темная)
- **Таблицы с сортировкой** - все колонки кликабельны
- **Графики** - интерактивные (Chart.js, Recharts)
- **Экспорт данных** - CSV, PDF
- **Быстрые действия** - горячие клавиши

## Метрики успеха
- Использование Studio > 80% авторов
- Среднее время в Studio > 10 мин/сессия
- Engagement с аналитикой > 60%
- Использование массовых действий > 30%
