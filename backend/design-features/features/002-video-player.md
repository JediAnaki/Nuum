# 002 - Видеоплеер

**Приоритет:** MVP (002)
**Категория:** Воспроизведение контента
**Зависимости:** Нет

## Описание

Полнофункциональный HTML5 видеоплеер с адаптивным стримингом, поддержкой различных качеств, субтитров и всех стандартных элементов управления. Основа просмотра контента на платформе.

## Функциональные требования

### 1. Базовое воспроизведение

#### 1.1 Основные элементы управления
- **Play/Pause** - пробел или клик по видео
- **Перемотка** - клик по прогресс-бару, стрелки влево/вправо (5 сек)
- **Громкость** - колесико мыши, стрелки вверх/вниз
- **Полноэкранный режим** - F или кнопка, двойной клик
- **Скорость воспроизведения** - 0.25x, 0.5x, 0.75x, 1x, 1.25x, 1.5x, 1.75x, 2x
- **Картинка в картинке (PiP)** - отдельная кнопка

#### 1.2 Прогресс-бар
- Показ текущего времени и общей длительности
- Превью кадра при наведении (thumbnail preview)
- Буфер загрузки
- Красная полоса просмотренного
- Желтые маркеры глав (если есть)
- Зеленые маркеры рекламы (пропускаемой части)

#### 1.3 Автовоспроизведение
- Переключатель в настройках плеера
- Сохранение предпочтения пользователя
- Автопереход к следующему видео из плейлиста
- Таймер обратного отсчета (5 сек) перед автопереходом
- Кнопка отмены автоперехода

### 2. Качество видео

#### 2.1 Адаптивное качество (HLS/DASH)
- **Auto** - автоматический выбор на основе скорости
- **144p** - минимальное (мобильный интернет)
- **240p** - низкое
- **360p** - среднее
- **480p** - SD
- **720p** - HD
- **1080p** - Full HD
- **1440p** - 2K (если доступно)
- **2160p** - 4K (если доступно)

#### 2.2 Индикация качества
- Текущее качество в правом верхнем углу плеера
- Меню выбора качества
- Индикатор "HD" badge
- Предупреждение о низкой скорости
- Плавное переключение без остановки

#### 2.3 Настройки качества
- Предпочтительное качество в настройках аккаунта
- "Экономия трафика" режим (только на мобильных)
- Ограничение качества на мобильном интернете
- Статистика потребления трафика

### 3. Субтитры и звуковые дорожки

#### 3.1 Субтитры
- Форматы: WebVTT, SRT
- Автоматические субтитры (если сгенерированы)
- Пользовательские субтитры на разных языках
- Позиционирование поверх видео
- Стилизация субтитров:
  - Размер шрифта (маленький, средний, больший)
  - Цвет текста (белый, желтый, черный)
  - Фон (прозрачный, полупрозрачный, непрозрачный)
  - Обводка текста
  - Семейство шрифта
- Сохранение предпочтений

#### 3.2 Звуковые дорожки
- Переключение между дорожками (русский, английский и т.д.)
- Индикация текущей дорожки
- Сохранение выбранного языка

### 4. Элементы управления

#### 4.1 Нижняя панель управления
```
[Play/Pause] [Следующее видео] [Громкость + слайдер] [00:00 / 10:25] =============[Прогресс]============= [Субтитры] [Настройки] [Кинотеатр] [Полный экран]
```

- Автоскрытие через 3 секунды бездействия
- Появление при движении мыши
- Всегда видна на паузе
- Градиент снизу для читаемости

#### 4.2 Верхняя панель (при наведении)
```
[Название видео]                                                    [Качество: 1080p]
```

#### 4.3 Центральные элементы
- Большая кнопка Play при паузе
- Иконки перемотки (двойной клик слева/справа)
- Анимация перемотки ±10 сек
- Индикатор буферизации (спиннер)
- Gesture controls на мобильных (свайп вверх/вниз для громкости)

### 5. Горячие клавиши

```
Пробел / K       - Play/Pause
→                - +5 секунд
←                - -5 секунд
L                - +10 секунд
J                - -10 секунд
↑                - Громкость +5%
↓                - Громкость -5%
M                - Mute/Unmute
F                - Полный экран
T                - Режим кинотеатра
I                - Мини-плеер (PiP)
C                - Субтитры вкл/выкл
< (Shift + ,)    - Замедлить
> (Shift + .)    - Ускорить
0-9              - Перейти к % видео (0% - 90%)
Home             - Начало видео
End              - Конец видео
```

### 6. Режимы просмотра

#### 6.1 Обычный режим
- Плеер 16:9 с рекомендациями справа
- Информация о видео под плеером
- Комментарии ниже

#### 6.2 Режим кинотеатра (Theater mode)
- Расширение плеера на всю ширину
- Затемнение фона
- Комментарии под видео
- Сохранение режима между видео

#### 6.3 Полноэкранный режим
- Нативный fullscreen API
- Поворот на мобильных устройствах
- Все элементы управления доступны
- ESC для выхода

#### 6.4 Картинка в картинке (PiP)
- Плавающее окно поверх других вкладок
- Изменение размера
- Перемещение в углы экрана
- Продолжение при переходе на другую страницу

### 7. Производительность

#### 7.1 Загрузка и буферизация
- Preload metadata до клика
- Адаптивная буферизация (1-5 сек вперед)
- Индикация буферизации
- Восстановление после потери соединения

#### 7.2 Оптимизация
- Ленивая загрузка плеера
- CDN для стриминга
- HLS.js для адаптивного стриминга
- Отключение качества недоступного при bandwidth

### 8. Аналитика просмотров

#### 8.1 Отслеживание
- Время начала просмотра
- Процент просмотренного
- Моменты паузы/перемотки
- Выбранное качество
- Устройство и браузер
- Среднее время просмотра

#### 8.2 Метрики
- Количество полных просмотров (>90%)
- Среднее удержание аудитории
- Моменты оттока зрителей
- Heat map популярных моментов

## UI/UX требования

### Дизайн плеера

### Адаптивность

#### Мобильная версия (<768px)
- Увеличенные кнопки (48x48px минимум)
- Упрощенная панель управления
- Свайп-жесты:
  - Двойной тап слева - перемотка назад
  - Двойной тап справа - перемотка вперед
  - Свайп вверх справа - увеличить громкость
  - Свайп вниз справа - уменьшить громкость
  - Свайп вверх слева - увеличить яркость
  - Свайп вниз слева - уменьшить яркость
- Автоповорот в landscape
- Настройки в модальном окне
- Превью кадров отключены (экономия трафика)

#### Планшет (768px - 1024px)
- Средний размер элементов
- Сенсорные жесты + клавиатура
- Превью кадров включены

#### Десктоп (>1024px)
- Все функции доступны
- Hover эффекты
- Превью кадров с высоким разрешением

### Структура компонента

```typescript
interface VideoPlayerProps {
  videoId: string;
  src: string | string[]; // URL или массив качеств
  poster?: string;
  autoplay?: boolean;
  startTime?: number; // продолжить с момента
  subtitles?: Subtitle[];
  chapters?: Chapter[];
  onProgress?: (time: number) => void;
  onEnded?: () => void;
  onQualityChange?: (quality: string) => void;
}

interface Subtitle {
  label: string; // "Русский", "English"
  language: string; // "ru", "en"
  src: string; // URL к .vtt файлу
  default?: boolean;
}

interface Chapter {
  time: number; // секунды
  title: string;
  thumbnail?: string;
}
```

## Обработка ошибок

### Типы ошибок

1. **Сеть недоступна**
   - "Нет подключения к интернету"
   - Кнопка "Повторить попытку"
   - Автоповтор каждые 5 сек

2. **Видео не найдено (404)**
   - "Это видео недоступно"
   - Возможные причины (удалено, приватное, заблокировано)
   - Кнопка "Вернуться на главную"

3. **Формат не поддерживается**
   - "Ваш браузер не поддерживает данный формат"
   - Предложение обновить браузер
   - Ссылка на список поддерживаемых браузеров

4. **Медленное соединение**
   - Toast: "Медленное соединение. Качество снижено до 360p"
   - Автоматическое понижение качества
   - Возможность выбрать качество вручную

5. **Ошибка воспроизведения**
   - "Произошла ошибка. Попробуйте перезагрузить страницу"
   - Код ошибки для поддержки
   - Кнопка "Перезагрузить плеер"

## Метрики успеха

- Время загрузки плеера < 1 сек
- Начало воспроизведения < 2 сек
- Буферизация < 5% времени просмотра
- Переключение качества < 500ms
- 0 ошибок воспроизведения для 95% пользователей
- Удержание > 60% в первые 30 секунд

## Особенности для русскоязычной аудитории

- Русский язык субтитров по умолчанию
- Русские названия скоростей ("Обычная", "Быстрее")
- Подсказки клавиш на русском
- Приоритет русских звуковых дорожек
- Формат времени 24-часовой
