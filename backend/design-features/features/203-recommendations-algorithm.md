# 203 - Продвинутый алгоритм рекомендаций

**Приоритет:** Средний (203)
**Категория:** ML и персонализация
**Зависимости:** 003 (Главная), 006 (Поиск)

## Описание
Улучшенный алгоритм рекомендаций с машинным обучением для максимальной релевантности.

## Факторы ранжирования

### Персонализация (60%)
- История просмотров
- Полностью просмотренные
- Лайки и дизлайки
- Подписки
- Поисковые запросы
- Клики но не досмотры (негативный сигнал)

### Качество видео (20%)
- CTR (Click-Through Rate)
- Average View Duration
- Watch time
- Engagement rate
- Retention curve

### Свежесть (10%)
- Дата публикации
- Тренды
- Сезонность

### Разнообразие (10%)
- Новые каналы для пользователя
- Разные категории
- Разная длина видео

## Модели ML

### Collaborative Filtering
- User-based
- Item-based
- Matrix Factorization

### Content-Based
- Анализ тегов, описания
- Визуальное сходство (thumbnails)
- Аудио fingerprinting

### Deep Learning
- Neural networks для предсказания CTR
- RNN для последовательностей просмотров
- Embeddings видео и пользователей

## A/B тестирование
- Постоянное тестирование алгоритмов
- Метрики: watch time, session time, retention

## Feedback loops
- "Не интересует" → негативный сигнал
- "Не рекомендовать канал" → блокировка
- Explicit feedback учитывается
