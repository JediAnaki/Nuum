# 205 - Расширенная аналитика

**Приоритет:** Средний (205)
**Зависимости:** 100 (Studio)

## Углубленные метрики
- Источники трафика (детально)
- География зрителей (карта)
- Демография (возраст, пол)
- Устройства и платформы
- Время просмотра по времени суток
- Удержание аудитории (посекундно)
- Click-through rate на превью
- Конверсия в подписчиков

## Сравнительная аналитика
- Сравнение с предыдущим периодом
- Сравнение видео между собой
- Бенчмарки индустрии

## Экспорт
- CSV, Excel
- PDF отчеты
- API доступ к данным
