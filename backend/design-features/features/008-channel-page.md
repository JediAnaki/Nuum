# 008 - Страница канала

**Приоритет:** MVP (008)
**Категория:** Каналы и контент
**Зависимости:** 001 (Авторизация), 003 (Главная страница)

## Описание

Публичная страница канала автора с видео, плейлистами, информацией о канале и возможностью подписки.

## Функциональные требования

### 1. Структура страницы канала

**URL:** `/@channelname` или `/channel/{channel_id}`

### 2. Шапка канала

```
┌────────────────────────────────────────────────┐
│ [════════════ Баннер 2560x1440 ═══════════════]│
├────────────────────────────────────────────────┤
│ [Avatar  ] НАЗВАНИЕ КАНАЛА ✓                   │
│ [120x120]  @channelname                        │
│            1.2M подписчиков • 500 видео        │
│            [Подписаться] [🔔] [✉️ Написать]    │
└────────────────────────────────────────────────┘
```

**Элементы:**
- **Баннер** - 2560x1440px (опциональный)
- **Аватар канала** - 120x120px, круглый
- **Название** - с галочкой верификации
- **Username** - @channelname (уникальный)
- **Статистика** - подписчики, количество видео
- **Кнопка подписки** - или "Вы подписаны"
- **Уведомления** - 🔔 настройка уведомлений
- **Написать** - отправить сообщение (если доступно)

### 3. Навигация канала

**Вкладки:**
```
[ Главная ] [ Видео ] [ Shorts ] [ Плейлисты ] [ Сообщество ] [ О канале ]
```

#### 3.1 Вкладка "Главная"
- Трейлер канала (для не подписанных)
- Популярные видео
- Последние загрузки
- Закрепленное видео
- Разделы (подборки видео)

#### 3.2 Вкладка "Видео"
```
[Последние ▼] [Поиск по каналу: _______]

[Видео карточка] [Видео карточка] [Видео карточка]
[Видео карточка] [Видео карточка] [Видео карточка]
```

**Сортировка:**
- Последние
- Популярные
- Старые

**Функции:**
- Поиск по видео канала
- Фильтр по годам/месяцам
- Бесконечная прокрутка

#### 3.3 Вкладка "Shorts"
- Вертикальные короткие видео
- Сетка 3-4 колонки
- Автопроигрывание при hover

#### 3.4 Вкладка "Плейлисты"
```
Созданные плейлисты

[Плейлист 1      ] [Плейлист 2      ]
[12 видео        ] [5 видео         ]
[▶ Воспроизвести] [▶ Воспроизвести]
```

#### 3.5 Вкладка "Сообщество" (посты)
- Текстовые посты
- Опросы
- Изображения
- GIF
- Лайки и комментарии к постам

#### 3.6 Вкладка "О канале"
```
Описание:
[Полное описание канала...]

Статистика:
• Зарегистрирован: 15 июня 2020 г.
• Всего просмотров: 150,234,567
• Страна: Россия

Ссылки:
• https://example.com
• https://instagram.com/channel

Связаться:
business@example.com
```

### 4. Подписка на канал

**Состояния кнопки:**

1. **Не подписан:**
```
[Подписаться]
```
- Красная кнопка
- Клик → подписка + анимация

2. **Подписан:**
```
[✓ Вы подписаны ▼]
```
- Серая кнопка с галочкой
- Dropdown:
  - 🔔 Все уведомления
  - 🔕 Персонализированные
  - 🚫 Не уведомлять
  - ❌ Отписаться

### 5. Кастомизация канала (для владельца)

**URL:** `/studio/customization`

**Секции:**

#### 5.1 Брендинг
- **Аватар** - 800x800px, круглый crop
- **Баннер** - 2560x1440px, safe area 1546x423px
- **Водяной знак** - для видео, PNG с прозрачностью

#### 5.2 Макет
- **Трейлер канала** - для не подписанных
- **Популярное видео** - для подписанных
- **Секции** - подборки видео, плейлисты

#### 5.3 Основная информация
- **Название канала** - до 100 символов
- **Описание** - до 5000 символов
- **Ссылки** - до 14 ссылок с названиями
- **Email** - для связи
- **Страна** - геолокация

## UI/UX требования

### Адаптивность

**Мобильный:**
```
[Аватар] Название
         @username
         1.2M подписчиков
[Подписаться] [⋮]

[Главная] [Видео] [О канале] →

[Видео]
[Видео]
```

**Планшет:**
- Уменьшенный баннер
- 2-3 колонки видео

**Десктоп:**
- Полный баннер
- 4-5 колонок видео
- Sidebar с информацией (опционально)

### Анимации
- Плавное появление баннера
- Fade-in аватара
- Анимация подписки (конфетти)
- Skeleton loaders при загрузке

## Технические детали

### API endpoints
```typescript
GET /api/v1/channels/{id}
GET /api/v1/channels/@{username}
PATCH /api/v1/channels/{id} // только владелец

GET /api/v1/channels/{id}/videos
GET /api/v1/channels/{id}/playlists
GET /api/v1/channels/{id}/community

POST /api/v1/channels/{id}/subscribe
DELETE /api/v1/channels/{id}/subscribe
PATCH /api/v1/channels/{id}/notification-settings

// Кастомизация
PATCH /api/v1/channels/{id}/branding
POST /api/v1/channels/{id}/banner
POST /api/v1/channels/{id}/avatar
```

### Типы данных
```typescript
interface Channel {
  id: string;
  username: string;
  name: string;
  description: string;
  avatar: string;
  banner?: string;
  is_verified: boolean;
  subscriber_count: number;
  video_count: number;
  total_views: number;
  created_at: string;
  country?: string;
  links: ChannelLink[];
  contact_email?: string;
  is_subscribed?: boolean;
  notification_level?: "all" | "personalized" | "none";
}

interface ChannelLink {
  title: string;
  url: string;
  icon?: string;
}
```

## Метрики успеха
- CTR на подписку > 8%
- Среднее время на канале > 5 мин
- Bounce rate < 50%
- Видео на посещение > 2
- Конверсия в просмотр видео > 70%

## Особенности для русскоязычной аудитории
- Кириллица в username
- Русские форматы дат
- Геолокация: Россия, Украина, Казахстан и др.
- Локальные социальные сети (VK, OK) в ссылках
