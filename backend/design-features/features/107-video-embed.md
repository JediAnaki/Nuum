# 107 - Встраивание видео (Embed)

**Приоритет:** Высокий (107)
**Категория:** Распространение контента
**Зависимости:** 002 (Плеер), 004 (Страница видео)

## Описание
Возможность встраивания видео на внешние сайты через iframe.

## URL
`/embed/{video_id}`

## Функции встраивания

### Генератор кода
На странице видео кнопка "Поделиться" → "Встроить":
```html
<iframe
  width="560"
  height="315"
  src="https://platform.ru/embed/abc123"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen
></iframe>
```

### Параметры URL
```
/embed/{id}?autoplay=1&start=60&mute=1&controls=0
```

**Доступные параметры:**
- `autoplay` (0/1) - автовоспроизведение
- `start` (секунды) - начать с момента
- `end` (секунды) - закончить на моменте
- `mute` (0/1) - без звука
- `controls` (0/1) - показать элементы управления
- `loop` (0/1) - зациклить
- `modestbranding` (0/1) - минимум брендинга
- `rel` (0/1) - показывать похожие в конце

### Плеер для embed
- Упрощенный интерфейс
- Только основные элементы управления
- Ссылка "Смотреть на платформе"
- Водяной знак канала (если настроен)
- Responsive (подстраивается под размер)

## Настройки автора
- Разрешить/запретить встраивание
- Список разрешенных доменов (whitelist)
- Показывать название и автора
- Показывать похожие видео

## Аналитика
- Просмотры из embed учитываются
- Источник трафика "Embedded"
- Топ сайтов по встраиванию

## Безопасность
- CORS настройки
- Защита от кликджекинга
- Rate limiting на embed запросы
- Защита от хотлинкинга

## API
```typescript
GET /api/v1/videos/{id}/embed-code
GET /api/v1/videos/{id}/embed-settings
PATCH /api/v1/videos/{id}/embed-settings
```

## Метрики
- % видео с разрешенным embed > 80%
- Просмотров из embed > 15% общих
- Среднее время просмотра embed > 70% обычного
