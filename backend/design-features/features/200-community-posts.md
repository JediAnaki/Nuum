# 200 - Посты сообщества (Community Tab)

**Приоритет:** Средний (200)
**Категория:** Социальное взаимодействие
**Зависимости:** 008 (Канал)

## Описание
Вкладка "Сообщество" на канале для текстовых постов, опросов, изображений и взаимодействия с подписчиками.

## Типы постов
- Текстовые посты (до 5000 символов)
- Изображения и GIF
- Опросы (до 4 вариантов)
- Видео-тизеры

## Взаимодействие
- Лайки постов
- Комментарии к постам
- Поделиться постом

## Создание поста
`/studio/community/create`

## API
```typescript
GET /api/v1/channels/{id}/posts
POST /api/v1/channels/{id}/posts
PATCH /api/v1/posts/{id}
DELETE /api/v1/posts/{id}
```
