# 202 - Встроенный видеоредактор

**Приоритет:** Средний (202)
**Категория:** Создание контента
**Зависимости:** 005 (Загрузка)

## Описание
Базовый онлайн-редактор для обработки видео после загрузки.

## Функции

### Обрезка
- Trim начала и конца
- Вырезать части из середины
- Split на несколько частей

### Улучшения
- Автоулучшение (стабилизация, яркость)
- Цветокоррекция (яркость, контраст, насыщенность)
- Фильтры (пресеты)

### Аудио
- Регулировка громкости
- Наложение музыки из библиотеки
- Удаление оригинального аудио
- Fade in/out

### Размытие
- Размытие лиц (автоматическое)
- Размытие областей (вручную)
- Удаление чувствительной информации

### Превью
- Генерация custom thumbnails
- Выбор кадра из видео
- Загрузка своего изображения

## Интерфейс
`/studio/editor/{video_id}`

Timeline с drag & drop
Превью в реальном времени
