# 003 - Главная страница и лента рекомендаций

**Приоритет:** MVP (003)
**Категория:** Контент и навигация
**Зависимости:** 001 (Авторизация), 002 (Видеоплеер)

## Описание

Главная страница платформы с персонализированными рекомендациями видео. Основная точка входа для пользователей и место обнаружения нового контента.

## Функциональные требования

### 1. Структура главной страницы

#### 1.1 Шапка сайта (Header)
**Левая часть:**
- Кнопка меню (бургер) - открывает боковую панель
- Логотип платформы - клик ведет на главную
- Название платформы (опционально на десктопе)

**Центр:**
- Поисковая строка с иконкой поиска
- Кнопка голосового поиска
- Placeholder: "Поиск видео..."
- История поиска (dropdown при фокусе)
- Автодополнение в реальном времени

**Правая часть:**
- Кнопка "Загрузить видео" (плюс в кружке)
- Иконка уведомлений с badge счетчика
- Аватар пользователя с dropdown меню
- Или кнопки "Войти" / "Регистрация" (для незалогиненных)

#### 1.2 Боковая панель (Sidebar)
**Основная навигация:**
- 🏠 Главная (активна)

- [Пользовательские плейлисты...]

**Разделитель**

**Подписки (топ-10):**
- [Аватар] Название канала
- ...

**Разделитель**

**Категории:**
- 🎮 Игры
- 🎵 Музыка
- ⚽ Спорт
- 🎬 Фильмы и ТВ
- 📰 Новости
- 🎓 Образование

**Футер:**
- Ссылки: О проекте, Пресса, Авторское право
- Правила, Конфиденциальность, Условия

#### 1.3 Контентная область

**Фильтры (горизонтальная прокрутка):**
```
[ Все ] [ Игры ] [ Музыка ] [ Прямые эфиры ] [ Новости ] [ Комедия ] [ Технологии ] ...
```
- Sticky при прокрутке
- Кнопки стрелок для прокрутки фильтров
- Выбранный фильтр подсвечен
- Динамическая подгрузка категорий

**Лента видео:**
- Сетка видео-карточек
- Адаптивная:
  - Мобильный: 1 колонка
  - Планшет: 2-3 колонки
  - Десктоп: 4-5 колонок
  - Широкий экран: 6+ колонок
- Бесконечная прокрутка (infinite scroll)
- Skeleton loaders при загрузке

### 2. Видео-карточка

#### 2.1 Структура карточки

**Превью (Thumbnail):**
- Соотношение 16:9
- Изображение в высоком качестве
- Длительность видео (правый нижний угол)
  - Формат: "12:34" или "1:23:45"
  - Темный фон с прозрачностью
- Progress bar просмотренного (внизу превью, красная полоса)
- Badge качества ("HD", "4K", "LIVE") в левом верхнем углу
- Hover эффект:
  - Затемнение
  - Появление кнопок: "Смотреть позже", "В плейлист"
  - Автопревью видео (GIF или короткий клип) через 1 сек

**Информация под превью:**
```
[Аватар канала (40x40px)]  [Название видео (2 строки max)]
                           [Название канала]
                           [100 тыс. просмотров • 2 дня назад]
```

**Детали:**
- Аватар канала (круглый, кликабельный)
- Название видео:
  - Максимум 2 строки
  - Многоточие при переполнении
  - Tooltip с полным названием при hover
  - Bold шрифт
- Название канала (серый цвет, кликабельное)
- Метаданные:
  - Количество просмотров (сокращенное: 1.2М, 543 тыс.)
  - Время публикации (относительное: "2 дня назад", "3 месяца назад")
  - Разделитель: точка

**Меню карточки:**
- Три точки (вертикальные) справа от названия
- Dropdown при клике:
  - ⏰ Смотреть позже
  - ➕ Добавить в плейлист
  - 🚫 Не интересует
  - 🚨 Пожаловаться
  - 📤 Поделиться
  - (Если свое видео):
    - ✏️ Изменить
    - 📊 Статистика
    - 🗑 Удалить

#### 2.2 Интерактивность

**Клики:**
- По превью → открыть страницу видео
- По названию → открыть страницу видео
- По аватару → перейти на канал
- По названию канала → перейти на канал

**Hover эффекты:**
- Плавное появление overlay
- Подсветка превью
- Изменение курсора на pointer
- Показ автопревью (опционально)

**Жесты (мобильные):**
- Свайп влево на карточке → меню действий
- Долгое нажатие → контекстное меню

### 3. Алгоритм рекомендаций

#### 3.1 Для авторизованных пользователей

**Источники данных:**
1. История просмотров (вес: 40%)
   - Последние 100 просмотров
   - Категории и теги
   - Полностью просмотренные видео (выше вес)

2. Подписки (вес: 30%)
   - Новые видео от подписок
   - Похожие каналы

3. Лайки и взаимодействия (вес: 15%)
   - Лайкнутые видео
   - Комментарии
   - Сохраненные в плейлисты

4. Поведенческие факторы (вес: 10%)
   - Время суток (ночью - развлечения, днем - обучение)
   - День недели
   - Длительность сессии

5. Трендовость (вес: 5%)
   - Популярное в вашем регионе
   - Вирусные видео

**Фильтрация:**
- Уже просмотренные видео исключаются (опция в настройках)
- Заблокированные каналы не показываются
- Неинтересующие категории (пользователь отметил)

**Разнообразие:**
- Не более 2-3 видео подряд от одного канала
- Микс категорий и форматов
- Иногда вставка новых для пользователя каналов

#### 3.2 Для неавторизованных пользователей

**Источники:**
1. Популярное сегодня (50%)
2. Популярное за неделю (30%)
3. Популярные категории (20%)

**Персонализация:**
- На основе региона (IP)
- На основе языка браузера
- Куки сессии (анонимная история)

### 7. Фильтры и сортировка

#### 7.1 Горизонтальные фильтры
```
[ Все ] [ Игры ] [ Музыка ] [ Живое ] [ Недавно загруженные ] [ Для вас ] ...
```

**Поведение:**
- Sticky при прокрутке вниз
- Горизонтальная прокрутка с кнопками
- Выбор фильтра обновляет ленту
- URL обновляется (?filter=games)
- История браузера сохраняется

**Специальные фильтры:**

## Технические детали

### API endpoints

```typescript
// Получение ленты
GET /api/v1/feed
Query params:
  - page: number (default: 1)
  - limit: number (default: 20, max: 50)
  - filter: string (all, gaming, music, live, etc.)
  - exclude_watched: boolean

Response: {
  videos: Video[],
  sections?: Section[], // именованные секции
  nextPage: number | null,
  hasMore: boolean
}

// Получение персонализированных рекомендаций
GET /api/v1/feed/recommendations
Query params:
  - count: number (default: 20)
  - seed_video_id?: string (для похожих)

Response: {
  videos: Video[],
  reason?: string // "Based on your watch history"
}

// Отметить "Не интересует"
POST /api/v1/feed/not-interested
Body: {
  video_id: string,
  reason?: string
}

// Получение трендов
GET /api/v1/trending
Query params:
  - category?: string
  - country?: string
```

### Типы данных

```typescript
interface Video {
  id: string;
  title: string;
  description: string;
  thumbnail: {
    url: string;
    width: number;
    height: number;
  };
  duration: number; // секунды
  channel: {
    id: string;
    name: string;
    avatar: string;
    verified: boolean;
  };
  stats: {
    views: number;
    likes: number;
    comments: number;
  };
  published_at: string; // ISO date
  is_live: boolean;
  is_premiere: boolean;
  is_short: boolean;
  quality_badges: ("HD" | "4K" | "VR")[];
  progress?: number; // 0-100, процент просмотренного
}

interface Section {
  id: string;
  title: string;
  type: "horizontal" | "grid";
  videos: Video[];
  show_more_url?: string;
}
```

## Метрики успеха

- Время загрузки первого экрана < 1.5 сек
- LCP (Largest Contentful Paint) < 2.5 сек
- FID (First Input Delay) < 100ms
- CLS (Cumulative Layout Shift) < 0.1
- CTR (Click-Through Rate) > 15%
- Среднее время на странице > 5 мин
- Bounce rate < 40%
- Видео на пользователя за сессию > 3

## Особенности для русскоязычной аудитории

- Склонение числительных ("1 просмотр", "2 просмотра", "5 просмотров")
- Относительные даты на русском ("вчера", "2 дня назад", "месяц назад")
- Приоритет русскоязычного контента для неавторизованных
- Региональные тренды (Россия, Украина, Казахстан и т.д.)
- Категории адаптированы под русский рынок
- Форматирование чисел с пробелами (100 000 вместо 100,000)
